This is the first post on edenjaeger.com. The site is built with
[Astro](https://astro.build) and hosted on Render.

Posts are just markdown files in `src/content/blog/`. To publish a new one,
add a `.md` file with frontmatter like this post's, push to GitHub, and the
site rebuilds automatically.

Set `draft: true` in the frontmatter to keep a post out of the site while
you work on it.
